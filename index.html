User Guide

Getting Started

  1. Connect your SPD-SX PRO via USB (set to Generic/Class-Compliant mode)
  2. Launch MIDI Viewer
  3. Open Settings (SET button, top right) and tap Scan
  4. Select the SPD-SX PRO from the device list
  5. Switch kits on the SPD — the display follows automatically

Importing Your Setup

  1. On your SPD-SX PRO: MENU → BACKUP → SAVE ALL → to USB flash drive
  2. Connect the USB flash drive to your device (iPad via USB-C adapter, Mac directly)
  3. In MIDI Viewer: Settings → Import Backup
  4. Select BKUP-ALL.TD0 (the ~4MB config file, not the larger .spd audio file)
  5. All 200 kit names, pad labels, triggers, and footswitches are imported

On iPad/iPhone you can also import from iCloud Drive or any source visible in the Files app.

Controls

SPD-SX PRO Setup

For MIDI Viewer to communicate with your SPD-SX PRO:

  • USB Driver Mode: set to GENERIC (not Vendor) — MENU → SYSTEM → USB DRIVER
  • PC TX/RX: set to ON — MENU → SYSTEM → MIDI
  • Connect via USB-C (iPad) or USB-A/B cable (Mac)

Troubleshooting

Device not foundCheck USB Driver is set to GENERIC. Try Settings → Scan again.
Kits show as emptyImport your backup file (TD0) — see Importing Your Setup above.
Import fails on iOSEnsure the TD0 file is fully downloaded (not an iCloud placeholder). Tap it in Files first.
Connection dropsThe SPD may revert to Vendor mode after power cycle. Set back to GENERIC.
Wrong pad labelsRe-import your backup after making changes on the SPD.
